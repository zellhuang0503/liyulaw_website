**【理宇法務常識專欄】**

**標題：不想留下前科？認識「緩起訴」與「認罪協商」：台灣刑事案件的兩條岔路，條件、程序、效果全解析 (2025年版)**

**前言：面對刑事案件，除了審判還有其他選擇嗎？**

「我被告了，會不會被關？」、「如果被判刑，是不是一輩子都有前科了？」、「有沒有可能大事化小，不要走到上法院那一步？」

當不幸涉及刑事案件，收到檢警通知或甚至被起訴時，隨之而來的往往是巨大的心理壓力、對未來的茫然，以及對「前科紀錄」的深切擔憂。許多人以為，刑事案件的結局只有兩種：不是無罪脫身，就是被判有罪留下紀錄。但事實上，在台灣的刑事訴訟制度中，針對特定類型的案件和被告，存在著幾種「替代性的處理方式」，讓案件有機會在進入最終審判前，以不同的方式畫下句點。

其中，**「緩起訴處分」和「認罪協商程序」是最為重要且常見的兩種機制。它們都旨在提高司法效率、節省訴訟資源，並在某些情況下，給予被告改過自新或獲得較輕處遇的機會。然而，這兩者是由不同的機關主導、在不同的訴訟階段發生、適用條件不同，法律效果（尤其是會不會留下前科**）更是有著**天壤之別**！

許多民眾常常將兩者混淆，或是對其內容一知半解，可能因此錯失維護自身權益的機會，或做出不符最佳利益的決定。今天，理宇法律事務所「法務常識」專欄，將為您深入淺出地介紹「緩起訴」與「認罪協商」這兩種制度，詳述它們的法律依據（主要在\*\*《刑事訴訟法》\*\*）、適用條件、進行程序、法律效力，以及最重要的——它們之間的關鍵差異。了解這兩條岔路，才能在面臨抉擇時，做出最明智的判斷。

**一、檢察官給的「暫緩」機會：深入認識「緩起訴處分」**

**（一）什麼是「緩起訴」？**

* **定義：** 「緩起訴處分」，規定在\*\*《刑事訴訟法》第 253 條之 1\*\*，是指**檢察官**在**偵查終結**後，雖然認為被告的犯罪嫌疑充足，但考量特定因素後，認為「暫緩起訴」為適當，於是設定一段\*\*「緩起訴期間」（1 年至 3 年）**，並要求被告在這段期間內遵守或履行某些**「條件」**。只要被告在緩起訴期間內**表現良好、確實履行條件\*\*，且緩起訴未被撤銷，那麼期滿後，這個案件就**等同於不起訴處分確定**，案件終結，被告**不會留下任何刑事前科紀錄**。  
* **主導者：** 作成緩起訴處分的權力在於\*\*「檢察官」\*\*。  
* **階段：** 發生在\*\*「偵查階段」\*\*，也就是檢察官決定是否要向法院提起公訴（起訴）**之前**。

**（二）緩起訴的目的與考量：**

* **微罪從寬，給予自新機會：** 主要適用於犯罪情節相對輕微、被告犯後態度良好、有悔意的情況，給予被告一個避免留下前科、重返社會的機會。  
* **節省司法資源：** 對於輕微案件，若都進入審判程序，將耗費大量司法資源。緩起訴有助於案件分流，讓司法資源集中處理重大案件。  
* **填補損害，促進和解：** 常見的緩起訴條件包含向被害人道歉、賠償損害或支付補償金，有助於修復犯罪造成的傷害，達到部分「修復式司法」的效果。  
* **預防再犯：** 要求被告完成戒癮治療、心理輔導等條件，有助於降低再犯風險。

**（三）誰有機會獲得緩起訴？（適用條件）**

並非所有案件都能獲得緩起訴。依據《刑事訴訟法》第 253 條之 1，必須符合以下條件：

1. **罪名限制：** 被告所犯之罪，其**法定最重本刑**必須**不是**死刑、無期徒刑 或 最輕本刑 3 年以上有期徒刑之罪。（換句話說，必須是相對較輕的罪名）。例如：過失傷害、普通竊盜、多數的公共危險罪（如部分酒駕）、偽造文書等，都可能是緩起訴的考慮範圍。  
2. **檢察官的裁量：** 檢察官會參酌《刑法》第 57 條所列的各種科刑因素（例如：犯罪動機、目的、手段、對被害人造成的損害、被告的品行、生活狀況、犯後態度等），以及公共利益的維護，綜合判斷後認為以「緩起訴」為適當。  
3. **被告的意願：** 雖然緩起訴的決定權在檢察官，但若要附加特定條件（如下述），通常需徵得被告的同意。

**（四）緩起訴的「條件」有哪些？（緩起訴期間應遵守或履行事項）**

檢察官在做出緩起訴處分時，**必須同時**要求被告在緩起訴期間（1 年至 3 年）內，遵守或履行\*\*《刑事訴訟法》第 253 條之 2\*\* 所列的**一項或數項**條件。常見的條件包括：

1. **向被害人道歉。**  
2. **立悔過書。**  
3. **向被害人支付「相當數額」之損害賠償。** （金額需合理）  
4. **向公庫或指定的公益團體、地方自治團體支付一定金額。** （俗稱「緩起訴處分金」）  
5. **向該管檢察署指定的政府機關、機構、行政法人、社區或其他符合公益目的之機構或團體提供「義務勞務」**（需在 40 小時以上，240 小時以下）。  
6. **完成「戒癮治療、精神治療、心理輔導」或其他預防再犯之處遇措施。**（例如：酒癮、毒癮、家暴防治等課程）  
7. **遵守保護被害人安全之必要命令。**（例如：禁止接觸被害人）  
8. **遵守預防再犯所為之必要命令。**（例如：禁止進入特定場所）

**（五）緩起訴的法律效果：成功與失敗的差別**

* **成功履行條件，期滿效果：** 如果被告在緩起訴期間內，確實遵守並完成了檢察官所指定的全部條件，並且沒有發生應被撤銷緩起訴之事由，那麼緩起訴期間一屆滿，緩起訴處分就**確定**了。其法律效果**等同於「不起訴處分確定」**，也就是：  
  * 該案件**完全終結**，檢察官不得再對同一案件提起公訴。  
  * 被告**不會有任何刑事判決紀錄**，也就是俗稱的\*\*「沒有前科」\*\*。  
* **未履行條件或違規的後果（撤銷緩起訴）：** 如果發生以下情況（《刑事訴訟法》第 253 條之 3），檢察官\*\*「得」\*\*（可以，通常也會）**依職權或依告訴人之聲請，「撤銷」原來的緩起訴處分**：  
  1. 在緩起訴期間內，**故意更犯**有期徒刑以上刑之罪，經檢察官提起公訴者。（注意：必須是**故意**犯罪，且被**起訴**）  
  2. 緩起訴前，**故意犯**他罪，而在緩起訴期間內**才被發現並起訴**者。  
  3. **未遵守或履行**檢察官所定之應遵守或履行事項（緩起訴條件），且**情節重大**者。（例如：無故不繳納處分金、不去義務勞務、不完成治療等）  
* **緩起訴被撤銷後：** 一旦緩起訴被撤銷，案件就會**回到原點**，檢察官會**繼續偵查或直接提起公訴**，被告就可能面臨被起訴、審判、甚至判刑的結果。之前為緩起訴所做的努力（如已支付的部分賠償金或勞務）並不能當然免除後續的法律責任，但可能成為法院量刑時的參考。

**二、法庭上的合意判決：深入認識「認罪協商程序」**

**（一）什麼是「認罪協商」？**

* **定義：** 「認罪協商程序」，規定在\*\*《刑事訴訟法》第三編第一章之一（第 455 條之 2 至第 455 條之 11）**，是指案件經**檢察官提起公訴（起訴）**或**聲請簡易判決處刑**後，在**法院**審理程序中（通常是第一審言詞辯論終結前或簡易判決處刑前），由**檢察官**和**被告\*\*（通常透過律師）就被告**承認犯罪（認罪）**，以及**科處的刑度（例如：判多久、是否緩刑）或應付的條件**（例如：罰金、道歉、賠償）進行**協商**。如果達成**合意**，就由檢察官向法院聲請，由**法官**裁量是否同意按照協商的內容來**下判決**。  
* **主導與參與者：** 由**檢察官**與**被告方**進行協商，最終結果需經\*\*「法官」\*\*同意。  
* **階段：** 發生在\*\*「審判階段」\*\*，也就是案件已經進入法院之後。

**（二）認罪協商的目的：**

* **簡化訴訟程序，提高效率：** 對於被告已認罪的案件，省略冗長的交互詰問、證據調查程序，快速終結案件。  
* **確保刑罰的相當性與預測性：** 讓被告在認罪的前提下，換取一個相對明確且可能較輕的刑罰結果；檢察官也能確保被告被定罪。  
* **鼓勵被告認罪悔改：** 提供誘因讓有悔意的被告承認犯行。

**（三）哪些案件可以進行認罪協商？（適用條件）**

認罪協商的適用範圍比緩起訴更為嚴格（《刑事訴訟法》第 455 條之 2）：

1. **罪名限制：**  
   * **被告所犯之罪，必須「不是」死刑、無期徒刑、最輕本刑 3 年以上有期徒刑之罪。**（與緩起訴相同）  
   * **且「不是」高等檢察署檢察長所指定的，不能進行協商的特定重大犯罪案件**（例如：某些貪污、重大經濟犯罪等可能被排除）。  
2. **訴訟階段：** 必須是案件已經由檢察官\*\*「提起公訴」**或**「聲請簡易判決處刑」\*\*之後。  
3. **被告「必須」認罪：** 這是認罪協商的**核心前提**。如果被告否認犯罪，就無法進行協商。  
4. **原則上須有律師代理：** 進行協商程序時，被告\*\*「應」**有**辯護人（律師）\*\*協助。除非法院認為被告沒有資力、案情簡單或有其他特殊情況，才能例外許可沒有律師代理。（《刑事訴訟法》第 455 條之 3）這是為了確保被告在協商過程中，充分了解權利義務，避免在資訊不對等下做出不利決定。

**（四）認罪協商的流程：**

1. **提出請求：** 可由**檢察官主動**徵詢被告意見，或由**被告（透過律師）向檢察官**表示協商意願。  
2. **進行協商：** 由檢察官與被告（及其律師）討論可能的判決結果，包含：  
   * **刑度範圍：** 例如，檢方同意向法院求處特定有期徒刑（不得超過法定刑）、拘役或罰金。  
   * **是否宣告緩刑：** 若符合緩刑要件（判 2 年以下徒刑、拘役或罰金），可協商請求法院宣告緩刑。  
   * **附加條件：** 可協商判決中要求被告向被害人道歉、支付賠償金、向公庫支付款項或提供義務勞務等（作為緩刑條件或判決主文內容）。  
3. **達成合意：** 若雙方達成共識，會製作**書面合意**。  
4. **聲請法院判決：** 檢察官持協商合意書，向法院聲請依此內容進行判決。  
5. **法院審查與裁定 (《刑事訴訟法》第 455 條之 4)：**  
   * 法官收到聲請後，會進行審查，確認：被告是否自願認罪？是否了解協商的後果？協商內容是否合法、公平、沒有顯失均衡？  
   * 若法官認為協商內容適當，就會**按照協商的合意內容下判決**。  
   * 若法官認為協商內容不當（例如：刑度太低顯失公平、違反法律規定、被告非自願認罪等），**可以駁回**檢察官的聲請，案件將**回復到正常的審判程序**。

**（五）認罪協商的法律效果：有罪判決，但可能較輕**

* **有罪判決與前科紀錄：** 透過認罪協商程序所獲得的判決，本質上仍然是一個\*\*「有罪判決」**。因此，被告**會有刑事前科紀錄\*\*。這是它與緩起訴最大的不同點！  
* **刑度的可預測性與可能減輕：** 雖然留有前科，但好處是判決的刑度（包含是否緩刑）是事先經過協商、雙方同意的，**可預測性高**，且通常會比經過完整審判後可能判處的刑度來得**輕**。  
* **判決的確定性：** 依據《刑事訴訟法》第 455 條之 10，對於依協商判決所科的刑，原則上**不得上訴**（除非有法律規定顯有錯誤等極少數例外）。這代表案件可以快速確定終結。

**三、緩起訴 vs. 認罪協商：關鍵差異一表看懂**

| 特徵 | 緩起訴 (Deferred Prosecution) | 認罪協商 (Plea Bargaining) |
| :---- | :---- | :---- |
| **決定者** | **檢察官** | **法官** (核准檢辯雙方合意) |
| **決定階段** | 偵查中，**起訴前** | **起訴後** (或聲請簡易判決處刑後) |
| **被告是否需認罪** | **否** (雖實務上多為認罪案件，但法律不要求) | **是** (必須認罪) |
| **程序啟動** | 檢察官職權發動 | 檢察官或被告(透過律師)請求，雙方合意後聲請法院 |
| **律師是否必要** | 否 (但強烈建議諮詢) | **原則上是** (保障被告權益) |
| **核心內容** | 暫緩起訴 \+ 附加條件履行 | 認罪 \+ 合意刑罰/條件 |
| **成功結果** | 期滿後效力同不起訴，案件終結，**無前科紀錄** | **有罪判決** (但刑度可能較輕/緩刑)，**有前科紀錄** |
| **失敗/撤銷後果** | 撤銷緩起訴，**繼續偵查或起訴** | 法院駁回協商，**回復正常審判程序** |

**四、我該如何選擇？給被告的考量點與提醒**

面對緩起訴或認罪協商的可能性時，被告應該考量哪些因素？

* **最在意的點：是否留下前科？** 如果「避免前科紀錄」是首要目標，那麼\*\*「緩起訴」\*\*是唯一可能的途徑（若符合條件且能成功履行）。「認罪協商」必定會留下前科。  
* **案件證據強度：** 如果檢方掌握的證據非常充分，定罪風險高，那麼認罪協商換取較輕刑度可能較為有利。若證據薄弱，則可能考慮爭取不起訴或無罪判決，而非緩起訴或協商。  
* **是否承認犯罪：** 認罪協商以「認罪」為前提。若堅持自己無罪，則此路不通。緩起訴雖不以認罪為法律要件，但實務上檢察官較可能對有悔意、承認部分事實的被告給予緩起訴。  
* **案件所處階段：** 緩起訴是在偵查階段由檢察官決定；認罪協商是在起訴後由法院決定。  
* **履行條件的能力與意願：** 緩起訴的條件（付款、勞務、治療等）是否能確實履行？認罪協商可能要求的賠償或緩刑條件是否能接受？  
* **被害人的態度：** 被害人是否願意和解、接受道歉或賠償，常是影響檢察官給予緩起訴或協商條件的重要因素。

**【最關鍵的提醒】：務必諮詢專業律師！**

無論是考慮爭取緩起訴，還是評估是否進行認罪協商，**尋求專業刑事律師的協助都至關重要！** 律師能夠：

* **評估案件證據與法律風險：** 判斷爭取緩起訴或進行協商的可行性與利弊。  
* **向檢察官爭取緩起訴：** 在偵查階段積極與檢察官溝通，提出有利事證與條件，提高獲得緩起訴的機會。  
* **代理進行認罪協商：** 在協商過程中，確保您的權利不受損害，爭取最有利的條件與刑度。  
* **解釋法律效果：** 讓您充分理解不同選擇的長短期後果（特別是前科問題）。  
* **確保程序合法：** 監督偵查或審判程序是否符合法律規定。

**切勿在不完全了解狀況或沒有法律專業人士協助下，輕易做出決定！**

「緩起訴處分」與「認罪協商程序」是台灣刑事訴訟法中，針對特定案件所設計的兩套重要的替代性處理機制。前者由檢察官主導，成功有機會免除前科；後者需經法官同意，雖會留下前科，但可能換得較輕的刑罰。

了解這兩種制度的差異、條件與效果，有助於被告在面臨刑事追訴時，能夠更全面地評估自身情況，做出符合自身最佳利益的選擇。然而，每個案件都是獨特的，法律的適用也充滿細節，自行判斷往往存在風險。
